# Teleprompyter*
Supply a script.txt or change the name of the file in the code

Press either left or right arrow keys, left or right mouse, backspace or space to move forward and back lines
Press escape to exit

Automatically adjusts to fullscreen and shows as many characters as possible, including next and previous line so that you can actually read onto the next line without it creating a massive gap

If there's any problems with it then open an issue or PR and I might even fix/merge it



_* See it's funny because there's no 'y' in teleprompter, but there is one in the title, because it's written in python_
